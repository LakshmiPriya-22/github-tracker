# github-tracker
A Python project that automates tracking of students’ GitHub activity. It fetches each student’s repositories, commit counts, and last commit date — then generates a PDF report for each student.
This project helps mentors or faculty easily monitor the progress of multiple students without manually checking GitHub profiles.
# Features

Accepts Registration Number or GitHub Username
Fetches public GitHub data using GitHub REST API
Displays:
1.Total repositories
2.Total commits
3.Last commit date
4.Generates a well-formatted PDF report
5.Clean, modular, and professional code structure


# Tech Stack

Python 3
GitHub REST API
Requests – API calls
ReportLab – PDF generation
